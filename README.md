# Donna Voice 🦋

Native iOS voice interface for Donna AI assistant.

## Features

- 🎙️ Real-time speech communication
- 🔒 Secure connection via mTLS
- 🗣️ On-device STT (Whisper)
- 🔊 Cloud TTS (ElevenLabs) with offline fallback
- 📱 Native iOS app (Swift/SwiftUI)

## Architecture

```
iPhone App ←→ VPS (WireGuard) ←→ Mac Mini (Donna)
```
